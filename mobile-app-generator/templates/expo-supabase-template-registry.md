## Tier 1: Production-Ready Full-Stack (Recommended)

### 1. Makerkit React Native Expo Turbo SaaS Kit
- **Repo:** https://github.com/makerkit/react-native-expo-turbo-saas-kit
- **Docs:** https://makerkit.dev/docs/react-native-supabase
- **License:** Free & Open Source
- **Stack:** Expo 52+, React Native, Tailwind CSS (NativeWind), Supabase, Turborepo, TypeScript
- **Features:**
  - Full auth system (email, social, magic link)
  - Supabase database + RLS
  - Supabase Storage
  - Monorepo architecture (Turborepo)
  - NativeWind styling
  - ESLint + Prettier configured
  - TypeScript strict mode
  - Infrastructure pre-configured
- **Best For:** SaaS apps, production apps, any serious mobile project
- **Rating:** S-Tier (Best free option)

- **Repo:** https://github.com/Razikus/supabase-nextjs-template

  - Full SaaS template (web + mobile)

  - RLS policies, task management demos
  - i18n support (EN/PL/ZH)
  - Legal documents included
  - Pre-built themes
- **Best For:** SaaS apps that need both web dashboard and mobile app
- **Rating:** S-Tier
