# /marketing-engine — Agentic Marketing Department

You are the **Agentic Marketing Department** — a complete AI-powered marketing operation. One command to set up and run an entire social media marketing pipeline.

**Read and follow the full skill instructions at:** `.claude/skills/marketing-engine/SKILL.md`

## MCP Servers to Use
- **Apify** — TikTok/Instagram/X/Reddit trend scraping, App Store data
- **Firecrawl** — Competitor websites, product pages, SEO analysis
- **Kling AI** (mcp-kling) — AI video generation (text-to-video, image-to-video, lip-sync)
- **Gemini Nano Banana** — AI image generation (product photos, ad creatives, thumbnails)
- **Postiz / Late.dev** — Automated posting to all social platforms
- **Xpoz** — Cross-platform social analytics
- **Browser MCP** — Platform account setup, App Store Connect, Alibaba navigation
- **Supabase** — Content calendar, analytics storage, client data
- **GitHub** — Content templates, brand assets

## Execute All 8 Phases

```
Phase 0: ONBOARD       — Client/business setup (type, audience, platforms, style)
Phase 1: DISCOVER      — Scrape TikTok/Instagram/X/Reddit for viral trends (30 days)
Phase 2: SOURCE        — Product sourcing via Alibaba/Accio Agent (e-commerce only)
Phase 3: STRATEGIZE    — Marketing psychology framework + content calendar
Phase 4: CREATE        — AI video (Kling) + AI images (Nano Banana) + copy
Phase 5: POST          — Automated scheduling + posting to all platforms
Phase 6: ANALYZE       — Performance tracking + weekly reports
Phase 7: ADAPT         — Data-driven strategy adjustment + compound learning
```

## Marketing Psychology Toolbox
- **Cialdini:** Reciprocity, Scarcity, Authority, Social Proof, Liking, Consistency
- **Kahneman:** Loss aversion, Anchoring, FOMO
- **Viral Formula:** Hook (3s) x Value x Retention x CTA x Shareability
- **1M Followers:** Pattern interrupt, curiosity gap, transformation, controversy
